# Lox

A C++ implementation of the jlox interpreter for the Lox language from *Crafting Interpreters* by Bob Nystrom

### Instructions

On Linux and MacOS, simply build with `make`. 

To run the REPL, run the program with no arguments:
`bin/lox`

To pass a Lox language file to the interpreter, pass the file location and name as an argument:
`bin/lox src/main.lox`
